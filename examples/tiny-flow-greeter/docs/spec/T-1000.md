# T-1000 - Greeter CLI Spec

## Objetivo
Implementar um microprojeto CLI que imprime saudacao personalizada.

## Escopo
- Criar modulo `src/greeter.py` com funcao `greet(name: str) -> str`.
- Criar entrada CLI opcional por argumento (`python -m src.greeter <nome>`).
- Cobrir comportamento com testes unitarios.

## Requisitos
1. `greet("Ana")` deve retornar `"Hello, Ana!"`.
2. Quando nome vazio/espacos, retornar `"Hello, world!"`.
3. CLI sem argumento deve imprimir `Hello, world!`.
4. CLI com argumento deve imprimir saudacao correspondente.

## Criterios de Aceitacao
- Testes unitarios cobrindo caminhos principal e fallback.
- Execucao da CLI sem erro em ambiente local.
